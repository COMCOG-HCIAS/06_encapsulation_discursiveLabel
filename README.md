# Encapsulation \& Discursive Labels: Reading time & Discourse particle analysis

## Files

- `1_descriptionReport`: outlier information and data description
- `2_mainReport`: original main report with models for reading time and discourse particle analysis
- `2_mainReport_with_AOI6`: main report with additional seperate model for AOI 6
- `2_mainReport_with_AOI-1-2-6`: main report with additional seperate model and p-values for AOIs 1, 2, and 6

## Dependencies

The reports use the helper package `DPKogHelpers` that is currently not publicly available. If you want to reproduce the analyses, please contact us and we will provide the package source code.
